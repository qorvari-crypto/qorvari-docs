# Qorvari Operational Organization Chart

This document outlines the corporate structure of Qorvari. With the exception of the CEO, every role represents a heavily constrained, specialized Artificial Intelligence Agent interacting on our GitHub Projects tracking board.

## Executive Suite
*   **Chief Executive Officer (CEO):** Solo Human Founder. The ultimate authority, holding executive execution override and budget deployment over the AI.
*   **Chief Agent Officer (CAO / Orchestrator):** The Central Router. Monitors GitHub webhooks, triggers subordinate agents based on Jira/Git states, and enforces FinOps token budgets.

## Product & Growth Division
*   **Chief Product Officer (CPO Agent):** Governs product direction and feature roadmaps.
*   **Chief Marketing Officer (CMO Agent):** Governs user acquisition, SEO, and developer advocacy.
*   **Head of HR & Performance (HR Agent):** Performs internal tuning and monitoring of the other agents' success rates and token expenditures.

## Engineering Division
*   **Chief Technology Officer (CTO Agent):** Governs architecture and code quality.
*   **VP of Backend Engineering:** Oversees Python/Java monolithic de-coupling and REST API logic.
*   **VP of Frontend Engineering:** Oversees the Angular SaaS Dashboard (`mtl-ui`).
*   **VP of Data / Chief DBA:** Oversees multi-tenant schema migrations.

## Operations & Security Division
*   **Chief Information Security Officer (CISO Agent):** Governs compliance, penetration testing, and leak prevention.
*   **VP of Infrastructure (DevOps Agent):** Packages the cloud staging environments and manages Terraform.
*   **VP of Technical Documentation:** Enforces IEEE/Industrial standard documentation across all engineers.
